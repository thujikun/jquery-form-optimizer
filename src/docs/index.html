jQuery Form Optimizer

This plugin enables to add validator and input helper your form!

Credit Card
     

HTML

<form id="sample-form1">
    <div class="input-table">
        <dl class="input-row">
            <dt>Required Field</dt>
            <dd>
                <input type="text" name="sample" data-validation='{"unit":{"required":true}}'>
            </dd>
        </dl>
    </div>
    <div class="input-table">
        <dl class="input-row">
            <dt>Credit Card</dt>
            <dd>
                <input type="text" name="sample1-credit1" class="short" maxlength="4" data-validation='{"relation":[{"type":"credit","key":"sample1-credit"}]}' data-input-helper='{"restriction":"number","autonext":4}'>
                 
                <input type="text" name="sample1-credit2" class="short" maxlength="4" data-validation='{"relation":[{"type":"credit","key":"sample1-credit"}]}' data-input-helper='{"restriction":"number","autonext":4,"autoprev":true}'>
                 
                <input type="text" name="sample1-credit3" class="short" maxlength="4" data-validation='{"relation":[{"type":"credit","key":"sample1-credit"}]}' data-input-helper='{"restriction":"number","autonext":4,"autoprev":true}'>
                 
                <input type="text" name="sample1-credit4" class="short" maxlength="4" data-validation='{"relation":[{"type":"credit","key":"sample1-credit"}]}' data-input-helper='{"restriction":"number","autoprev":true}'>
            </dd>
        </dl>
    </div>
</form>

Javascript

var $form = $('#sample-form1');

// initialize validator
$form.validator();
// initialize input helper
$form.find('[data-input-helper]').inputHelper();

validator unit

Unit Checks

# Check Type Key type Key Description
required Boolean true or false check inputted or not
required-check Number the number of check required check checkbox is checked more than specified value
length Number length check inputted value equals length or not
maxlength Number max length check inputted value is lower than maxlength or not
minlength Number min length check inputted value is higher than minlength or not
maxnum Number max number check inputted value is lower than max number or not
minnum Number min number check inputted value is higher than min number or not
format String number check inputted value is only number character or not
half-char check inputted value is only half character (number and alphabet) or not
half check inputted value is only half character (including symbol) or not
half-kana check inputted value is only half kana character or not
url check inputted value is only url format or not
tel check inputted value is only tel format or not
tel-i18n check inputted value is only international tel format or not
email check inputted value is only email format or not
full-kana check inputted value is only Full-width kana character or not
date check inputted value is only date format or not
dateYM check inputted value is only date(Year and Month) format or not
credit check inputted value is only credit format or not
regexp String regular expression check inputted value fit to specified regular expression or not

Relation Checks

# Check Type role Role description Description
same body target field Check the value is same between body and confirmation field.
confirmation email field for confirmation
date-around before previous date field Check privious date is smaller than latter date.
after latter date field
credit - - Check rune algorithm with multiple credit card number field.

# helper Type Key type Key Description
cancelspace Boolean true or false Ignore input space
autonext Number condition digit Auto focus to next item when user input specified digit
autoprev Boolean true or false Auto focus to prev item when user pressed backspace key and nothing is inputted
full2half Boolean true or false Convert Full-width alphabet and number to Half-width alphabet and number
fullKana2halfKana Boolean true or false Convert Full-width kana to Half-width kana
half2full Boolean true or false Convert Half-width alphabet and number to Full-width alphabet and number
halfKana2fullKana Boolean true or false Convert Half-width kana to Full-width kana
restriction String number restrict inputted value only width number
String half-char restrict inputted value with only Half-width alphabet and number
String half restrict inputted value with only Half-width alphabet, number and symbols
String tel restrict inputted value with only number and hyphen
String full-kana restrict inputted value with only Full-width kana
String zip restrict inputted value with only zip code
String date restrict inputted value with only number and slash
String credit restrict inputted value with only number and hyphen
insert String zip Insert hyphen automatically accourding to zip number format
String credit Insert hyphen automatically accourding to credit number format.
credit Boolean true or false Ditect credit card company form credit card number and trigger 'ditectcredit' event.
remove String Selector of remove target element Remove value when inputting something to the element.
